03 / Component plan

Magic UI + ReactBits는 과하게 넣되, 역할을 나눈다.

Magic UI: Particles / Animated Gradient Text 첫 인트로 배경과 제목에 사용. 여행 포스터 같은 첫 인상을 만들고, reduced-motion에서는 정적 gradient로 축소.
ReactBits: ScrollStack / ScrollFloat 타임라인 카드를 스크롤 카드 스택처럼 보여주고, 큰 카피는 살짝 parallax float 처리.
Motion + GSAP splash → app 전환은 Motion layout/scroll 기반, 배경/티켓의 미세 움직임은 GSAP timeline로 처리.
KTX Ticket UI 출발/도착/열차번호/호차/좌석을 boarding pass처럼 표시. 왕복 티켓은 swipe 또는 stacked card로 전환.
Kakao Place Card 각 일정 카드 하단에 Kakao Place 링크 액션. 카카오맵은 active itinerary와 동기화되는 마커를 표시.
Photo Marquee 준비 나중에 여행 사진이 생기면 Magic UI Marquee/ReactBits effect로 사진 띠를 추가. 지금은 일정 인트로만.
